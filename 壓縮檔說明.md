## 🚀 使用說明

### 1. 解壓縮
將 `強化學習遊戲平台_v1.0.zip` 解壓縮到任意目錄

### 2. 啟動平台

#### Windows 用戶
```bash
# 雙擊執行
start.bat
```

#### Linux/macOS 用戶
```bash
# 給予執行權限
chmod +x start.sh

# 執行啟動腳本
./start.sh
```

### 3. 訪問平台
啟動完成後，在瀏覽器中訪問：
- **主要地址**: http://localhost:8080
- **備用地址**: http://localhost:8000

### 4. 停止平台
```bash
# Windows
stop.bat

# Linux/macOS
./stop.sh
```
